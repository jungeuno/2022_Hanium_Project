# 팀원들 필수 수행 사항
1) branch 활용 철저히 할 것(개발 시작 전 master 인지 git branch 명령어를 통해 꼭 확인할 것)
2) 프로젝트에 수정사항이 있을 경우, 반드시 패치노트를 작성할 것
3) master branch 에 merge 할 경우, 반드시 카톡으로 팀원들에게 알릴 것
4) 멘티들은 확인 후 코멘트 작성
