# 파트별 개발 진행 사항 점검 & 추후 개발 계획
   - 마스크 유형 추천 -> 담당자 : 황유현
      # 진행 사항
      1) 테스트 데이터 셋 수집 후 모델 테스트 -> 테스트 결과 안좋았음
      2) 한장씩 예측결과 확인 중
      3) tflite 안드로이드 연결 링크 정리 중

      # 개발 계획
      1) 테스트 데이터 입력 수정 후 모델 테스트 재수행
      2) tflite 모델 연결 후 출력 결과 확인

   - 마스크 색상 추천 -> 담당자 : 강희정
     # 진행 사항
     1) 사진 업로드 버튼 구현 -> 실시간 인식은 어렵고 필요 X -> 사진 업로드 방식으로 구현 결정
         ㄴ 사진 촬영 버튼 & 갤러리 업로드 버튼 추가
         ㄴ 문제점 : 버튼이 너무 많아질 것 같음
     2) 진단 결과 창에 들어갈 내용 수집 중
     3) 시작 시 퍼스널 컬러, 얼굴형에 대한 중요성 설명 화면 추가 의견 제시
     4) 진단화면 의견 제시
         진단화면 - 얼굴인식 됨 -> 된거 알려주고 -> 진행
                     - 얼굴인식 안됨 -> 안됨 -> 잘 되기위한 설명

      # 개발 계획
      1) K-means 알고리즘 Kotlin 구현
      2) K-means 값 추출 이후 처리 구현
      3) 퍼스널 컬러 화면 디자인 구상
      4) 퍼스널 컬러 설명창에 들어갈 내용 정리(시안 3개 정도 제작 예정)
