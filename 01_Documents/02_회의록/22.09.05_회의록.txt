## 파트별 개발 진행 사항 점검 & 추후 개발 계획
   ### 마스크 유형 추천 -> 담당자 : 황유현
      진행 사항
      1) 결과화면 구현 완료

      개발 계획
      1) 계속 동일한 얼굴형 결과 출력 -> 이미지 입력 부분 확인
      2) 얼굴 부분에 사각형 그래픽 추가해서 얼굴 인식 여부 확인

   ### 마스크 색상 추천 -> 담당자 : 강희정
     진행 사항
     1) 마이페이지 화면 구현

     3) Bitmap to Array 코드 구현 중 에러 발생

     개발 계획
     1) 이미지 전처리 완료
     2) 퍼스널 컬러 결과값 추출(퍼스널컬러 구현 완료)
